# Finding an image for your pad
First step for your custom pad is finding an image. Search google or wallpaper sites or commision someone to make something for you.


Some sites for wallpapers are https://wallhaven.cc/ https://www.pinterest.com/ & artists on twitter.


## Finding orinigal images
If you've found an image from a non original source you ideally want the best possible resolution. Reverse image search will help you find the original one and you can work from that.
What i've found works best is googles reverse images search https://images.google.com/
